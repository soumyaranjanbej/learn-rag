RAG Pipeline Simulator

Watch a question become an answer, step by step.

Type a question about a small demo knowledge base and run it through every stage of Retrieval-Augmented Generation — chunking, embedding, retrieval, prompt augmentation, and generation — with the underlying math made visible at each step.

This runs entirely in your browser, so it can't call a real embedding model or LLM. Instead: embeddings are simulated with TF‑IDF vectors (word-frequency math you can actually see), and generation is simulated by extracting the most relevant sentences from the retrieved chunks rather than an LLM writing fluent prose. Real systems use dense neural embeddings (which also capture meaning for synonyms and related concepts, not just exact word overlap) and a real language model for generation — but every stage of the pipeline, and the logic connecting them, is accurate.
